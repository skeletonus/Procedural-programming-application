	В процедурном программировании каждая переменная имеет определённый тип данных, который указывает, какие значения она может хранить и какие операции над ними допустимы. В языке Python основными встроенными типами данных являются: целые числа, числа с плавающей точкой, строки и логический тип.

	Целые числа (int) — используются для хранения положительных и отрицательных чисел без дробной части. Например: age = 25. Над целыми числами можно выполнять арифметические операции: сложение, вычитание, умножение, деление и другие.

	Числа с плавающей точкой (float) — представляют дробные числа. Например: price = 19.99. Они также участвуют в арифметических вычислениях, но могут содержать небольшие погрешности из-за особенностей хранения в памяти.

	Строки (str) — последовательности символов, заключённые в кавычки. Например: name = "Мария". Строки нельзя складывать с числами напрямую, но их можно объединять между собой (конкатенация) или вставлять в f-строки. В Python строки заключаются в одинарные '' или двойные кавычки "". Разницы между ними нет, главное не использовать оба вида для одной строки

	Логический тип (bool) — может принимать только два значения: True (истина) или False (ложь). Этот тип часто используется в условиях (if) и циклах (while). Например: is_valid = True.

	Важно: при вводе данных с помощью input() всегда получается строка, даже если пользователь ввёл число. Чтобы преобразовать строку в число, используются функции int() и float(). Например:
	x = int(input("Введите число: ")).

	Особенность процедурного программирования — явная работа с типами данных внутри процедур: процедура может читать строку, преобразовывать её в число, выполнять вычисления и выводить результат. Понимание типов помогает избежать ошибок, таких как попытка сложить строку и число.
